# Student Registration Application

Full Stack CRUD Application using:

- React JS
- Bootstrap
- Node JS
- Express JS
- MySQL

## Features

- Register Student
- Get All Students
- Update Student
- Delete Student

## Backend Architecture

- Controller Layer
- Service Layer
- Repository Layer

## Setup

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm start

### Database

Import SQL file into MySQL
